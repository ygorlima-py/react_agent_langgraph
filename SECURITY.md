# Security Policy

## Supported Versions

Security fixes are provided only for the latest version on the `main` branch.

| Version / Branch | Supported |
| --- | --- |
| `main` | ✅ |
| Older tags / forks | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately.

Preferred method:
- Use **GitHub Security Advisories** (Security → Report a vulnerability)

Alternative:
- Email: **security@YOURDOMAIN.com** (replace with your email)

What to expect:
- We will acknowledge receipt within **72 hours**.
- If the report is confirmed, we will work on a fix and publish a security update.
- Please do not open public issues for security reports.
